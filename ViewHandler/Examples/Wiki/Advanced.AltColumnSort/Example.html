Demonstrates

Sorting a column on a different value than what's shown. In this example months are sorted chronologically instead of alphabetically.
In the cell data the value must be split up with the || separator.

Part 1: is visible ('January', 'February' etc)
Part 2: not used here
Part 3: value to sort on ('01', '02', etc)

Example: "startdate": "January||||01"

Show code 

if (configName === 'Example.config') {
  viewConfig.addDataStore('standard-data.json');
  viewConfig.addColumns([
    { title: 'startdate', altSort: true, sort: true}
  ]);
}
