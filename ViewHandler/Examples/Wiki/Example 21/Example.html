Demonstrates

Overriding the display value in a cell.
The 'country' column values are made into links.
The 'stardate' column values are changed to uppercase.

Show code 

viewConfig.convertDisplayValue = function (columnInfo, cfgValues) {
  if (columnInfo.getTitle() === 'country') {
    return cfgValues.display.toUpperCase();
  }
};
