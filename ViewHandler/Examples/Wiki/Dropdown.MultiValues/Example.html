Demonstrates

A cell can contain multiple filter values. In this example each entry contains one or several tags.
Each of them appears in the dropdown filter and can be filtered on individually.

Example data: "tags": "fantasy, dystopian||||fantasy, dystopian||fantasy_____dystopian"

Show code 

viewConfig.addColumns([
  { title : 'tags', type : COLUMN.DROPDOWN_MULTIPLE, altFilter : true, width: 170 }
]);
