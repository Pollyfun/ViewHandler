Demonstrates

Applying css-rules to columns, rows or a combination of rows and columns.

Show code

Configuration
viewConfig.addCss('Examples/Wiki/Advanced.CssRules/Example.css');
viewConfig.addColumns([
  { title: 'formating', type: COLUMN.CLASSES }
]);
Css
ul:not(.selected) li[column="name"] {
  background-color: blanchedalmond;
}
ul.green:not(:hover):not(.selected) li[column="country"] {
  background-color: darkgrey;
}
ul.red li[column="duration"] {
  font-weight: bold;
}
ul.yellow:not(:hover):not(.selected) li[column="duration"] {
  font-style: italic;
  background-color: yellow;
}
ul:not(:hover):not(.selected) li[column="startdate"] {
  background-color: red;
}
