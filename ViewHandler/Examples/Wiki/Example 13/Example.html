Demonstrates

Applying css-rules to columns, rows or a combination of rows and columns.

Configuration

  viewConfig.addColumns([
    { title: 'formating', type: COLUMN.CLASSES }
  ]);

Css

  li[itemname="name"] {
    background-color: blanchedalmond;
  }
  ul.green li[itemname="country"] {
    background-color: darkgrey;
  }
  ul.red li[itemname="duration"] {
    font-weight: bold;
  }
  ul.yellow li[itemname="duration"] {
    font-style: italic;
    background-color: yellow;
  }
  li[itemname="startdate"] {
    background-color: red;
  }
