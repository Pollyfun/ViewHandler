Demonstrates

Hierarchical dropdown filters. Single and multiple value dropdown filters can be mixed freely.

Configuration

  viewConfig.addColumns([
    { title: 'country', type: COLUMN.DROPDOWN_MULTIPLE },
    { title: 'formating', type: COLUMN.DROPDOWN },
    { title: 'startdate', type: COLUMN.DROPDOWN }
  ]);
