Demonstrates

Alternative dropdown sorting. Each entry in the dropdown filter contains both a value (hidden) and a text (visible).
The are separated by ||. For example: 07||July. See the data file for more illustration.
When altFilter is set to true, the dropdown sorts its entries by the hidden value.

Show code 

viewConfig.addColumns([
  { title: 'startdate', type: COLUMN.DROPDOWN, altSort: true, altFilter: true, numSort: true, sort: true }
]);
