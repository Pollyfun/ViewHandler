Demonstrates

The data in this column is not shown. Instead its added to the 'class' attribute in the ul-row. It can be used for css-layout.

Configuration

  viewConfig.addColumns([
    { title: 'formating', type: COLUMN.CLASSES }
  ]);

Css

  ul.yellow { background-color: yellow; }
  ul.red { background-color: red; }
  ul.green { background-color: green; }
